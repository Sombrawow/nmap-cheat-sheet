# Contributing to Nmap Cheat Sheet

¡Gracias por tu interés en contribuir! Este proyecto busca ser la guía definitiva de Nmap para profesionales de ciberseguridad.

## Cómo Contribuir

1. **Fork el repositorio**
2. **Crea una rama** (`git checkout -b feature/AmazingFeature`)
3. **Haz tus cambios**
4. **Commit** (`git commit -m 'Add some AmazingFeature'`)
5. **Push** (`git push origin feature/AmazingFeature`)
6. **Abre un Pull Request**

## Estándares de Código

- Usa ejemplos prácticos y reales
- Incluye explicaciones claras
- Verifica que los comandos funcionen correctamente
- Añade contexto sobre cuándo usar cada comando

## Reportar Issues

Si encuentras errores o tienes sugerencias, por favor abre un issue con:
- Descripción del problema/sugerencia
- Comandos afectados
- Contexto de uso
